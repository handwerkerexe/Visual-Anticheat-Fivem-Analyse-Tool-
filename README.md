# 🛡️ Visual Analyse

> Advanced FiveM Analysis & Detection System

Visual Analyse is a security and analysis solution designed for FiveM servers. Once started, the software continuously analyzes the environment and helps server owners identify suspicious activity and unauthorized software.

---

## 🇬🇧 English

### Features

- Continuous background analysis
- Detection of suspicious software
- Live monitoring while playing
- Configurable server actions
- Connection denial system
- Automatic ban system
- Detection logging
- Discord integration
- Lightweight and optimized

### How It Works

1. The player launches Visual Analyse.
2. The software starts monitoring in the background.
3. Suspicious activity is analyzed continuously.
4. If a violation is detected, configured actions can be executed.

Possible actions:

- Deny server connection
- Automatically ban the player
- Log detections
- Notify administrators

### Discord Integration

Server owners will be able to manage their Visual Analyse configuration through Discord.

**Discord:** `visualanticheat`

### Goal

Visual Analyse aims to help FiveM server owners maintain a fair environment by identifying suspicious activity and providing configurable response options.
